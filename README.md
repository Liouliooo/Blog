# 我的个人网站

## **我的博客**

我的[主页](https://www.liouliooo.com/)

## 使用方法

**先安装nodejs**

**然后执行**

```shell
$ git clone git@github.com:Liouliooo/Blog.git
$ cd Blog
$ npm install
```

## 部署方法

部署位置: github

**首先，用git config设置user.name和user.email**

**然后，设置ssh-key**

**最后，执行命令:**

``` shell
$ bash deploy.sh
```
